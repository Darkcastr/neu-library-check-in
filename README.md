# NEU Library Check-In System

## Overview
The NEU Library Check-In System is a web-based application that streamlines the library check-in and check-out process for NEU students, staff, teachers, and administrators. The system requires users to log in and specify their purpose for visiting the library, with all transactions being recorded for tracking and reporting purposes.

## Key Features
- **Fast & Reliable Login**: Quick and dependable authentication system
- **Purpose Tracking**: Records the reason for each library visit
- **Session Management**: Automatic tracking of check-in and check-out times
- **Multi-User Support**: Supports students, staff, teachers, and administrators
- **Secure Records**: All transactions are securely logged and stored

## Technology Stack
- **Frontend**: TypeScript, CSS
- **Backend**: PL/pgSQL
- **Database**: PostgreSQL
- **Architecture**: Web-based application

## How to Use
1. Visit the application: https://neu-library-buddy.lovable.app
2. Log in with your NEU credentials
3. Select your user type (Student/Staff/Teacher/Admin)
4. Enter your purpose for visiting the library
5. Check in to the system
6. Check out when you're done

## Target Users
- NEU Students
- NEU Staff/Employees
- NEU Teachers
- NEU Administrators

## Current Status
The application is fully functional as a school project and provides reliable library check-in/check-out capabilities.
